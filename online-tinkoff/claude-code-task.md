# Задача: Добавить юридические страницы для подключения эквайринга

## Контекст
Для подключения онлайн-оплаты (Тинькофф/ЮKassa) на сайт olai.art нужно добавить обязательные юридические документы.

## Что нужно сделать

### 1. Создать страницу /oferta (Публичная оферта)
- Создать файл `app/oferta/page.tsx`
- Использовать тот же layout что и остальные страницы
- Контент из файла `oferta.md` (прикреплён)
- Стилизация: простой читаемый текст, заголовки, отступы

### 2. Создать страницу /privacy (Политика конфиденциальности)  
- Создать файл `app/privacy/page.tsx`
- Контент из файла `privacy.md` (прикреплён)
- Аналогичная стилизация

### 3. Добавить ссылки в footer
В footer сайта добавить ссылки:
- Публичная оферта → /oferta
- Политика конфиденциальности → /privacy

### 4. Добавить реквизиты в footer
Добавить в footer (мелким шрифтом):
```
ИП Орлов Леонид Андреевич
ИНН 662342643820 | ОГРНИП 319665800217251
help@olai.art
```

### 5. Добавить чекбокс согласия в форму заказа (если есть)
Если на сайте есть форма заказа/оплаты, добавить чекбокс:
"Я согласен с [Публичной офертой](/oferta) и [Политикой конфиденциальности](/privacy)"

## Стек
- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Файлы с контентом
- oferta.md — текст оферты
- privacy.md — текст политики

## Критерии готовности
- [ ] Страница /oferta открывается и отображает текст оферты
- [ ] Страница /privacy открывается и отображает политику
- [ ] В footer есть ссылки на обе страницы
- [ ] В footer есть реквизиты ИП
- [ ] Страницы адаптивны (mobile-friendly)
